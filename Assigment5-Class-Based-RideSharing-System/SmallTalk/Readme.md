# RideShare Smalltalk Project

This project implements a simple ride-sharing system in **Pharo Smalltalk**, including the classes:

- `Ride` (abstract superclass)
- `StandardRide`
- `PremiumRide`
- `Driver`
- `Rider`

The package is exported as a single `.st` file (`RideShare.st`).  
A separate text file (`Test.txt`) contains example code to run and test the system from a Playground.

---

## Requirements
- **Pharo Smalltalk 11** or later  
- Transcript window (for program output)

---

## How to Load the Package

1. Open **Pharo**.
2. Drag and drop `RideShare.st` into the Pharo window  
   **OR**  
   go to `File → File in…` and select the file.
3. The classes will appear in the `RideShare` package in the System Browser.
